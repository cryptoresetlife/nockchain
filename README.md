# nockchain

## 使用方法：
* 下载后打开终端输入：
```bash
cd Downloads
```
添加执行权限：
```bash
chmod +x setup_nockchain.sh
```
执行：

```bash
./setup_nockchain.sh
```

等到5月19日开始后运行节点

```bash
cd nockchain
make run-nockchain-follower
```
